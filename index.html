LIGAMX Stats - Aplicación

Nota: Esta es una aplicación Flask que requiere un servidor Python para ejecutarse. GitHub Pages solo puede mostrar contenido estático.

Instrucciones para ejecutar localmente

Para ejecutar esta aplicación en tu computadora, sigue estos pasos:

  1. Clona el repositorio: 
    git clone https://github.com/juanrdz99/ligamx.git
  2. Navega al directorio del proyecto: 
    cd ligamx
  3. Instala las dependencias: 
    pip install -r requirements.txt
  4. Configura las variables de entorno en un archivo .env: 
    LIVESCORE_API_KEY=tu_api_key
LIVESCORE_API_SECRET=tu_api_secret
  5. Ejecuta la aplicación: 
    python app.py
  6. Abre tu navegador y visita: 
    http://localhost:5000

Características principales

  • Dashboard con estadísticas de la Liga MX
  • Monitoreo del sistema con tarjetas de estadísticas
  • Gráficos de tendencias para llamadas API, tasa de éxito, tiempo de respuesta y errores
  • Métricas de equipos y jugadores
  • Datos en tiempo real de la API de LiveScore
